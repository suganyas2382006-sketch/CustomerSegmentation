# sales_dashboard-
Sales & Revenue Analysis Dashboard

Project Overview

The Sales & Revenue Analysis Dashboard is a data analytics project developed using Python, Streamlit, Pandas, and Plotly. The dashboard helps businesses analyze sales performance, revenue trends, product performance, and regional sales distribution through interactive visualizations.

Objectives

- Analyze sales and revenue data.
- Track key business performance indicators (KPIs).
- Identify top-performing products.
- Monitor revenue trends over time.
- Visualize sales distribution by region and category.
- Generate business insights for decision-making.

Tools and Technologies

- Python
- Streamlit
- Pandas
- Plotly

Project Structure

SalesDashboard/

├── app.py

├── sales_data.csv

├── requirements.txt

└── README.md

Dataset

The dataset contains the following fields:

- Date
- Product
- Category
- Region
- Quantity
- Unit Price
- Revenue

Features

KPI Metrics

- Total Revenue
- Total Quantity Sold
- Total Orders

Visualizations

- Revenue Trend Line Chart
- Revenue by Product Bar Chart
- Revenue by Region Pie Chart
- Revenue by Category Bar Chart

Installation

Install the required libraries:

pip install -r requirements.txt

Running the Application

Run the following command:

streamlit run app.py

Results

The dashboard provides:

- Revenue analysis over time
- Product-wise revenue comparison
- Region-wise sales distribution
- Category-wise revenue analysis
- Business performance insights

Conclusion

This project demonstrates the use of Python for business analytics and dashboard development. It provides an interactive platform for analyzing sales and revenue data and supports data-driven decision-making.
