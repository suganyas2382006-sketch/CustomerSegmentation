# Customer Segmentation Project

## Objective
Segment customers based on purchasing behavior and demographics using K-Means Clustering.

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Plotly

## Features
- Customer clustering
- Purchase behavior analysis
- Interactive visualization
- Cluster summary statistics

## Algorithm
K-Means Clustering

## Outcome
Identified customer groups based on Annual Income and Spending Score to support targeted marketing strategies.
